# Esti Cotizaciones

Sistema profesional de cotizaciones desarrollado con Vue 3, Vite y Tailwind CSS.

## Características

- 🎨 Interfaz moderna y responsiva
- 📄 Generación de cotizaciones en PDF
- 🚀 Rendimiento optimizado con Vite
- 💪 TypeScript para código tipado
- 🎯 Componentes reutilizables

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar desarrollo
npm run dev

# Construir para producción
npm build

# Preview de la build
npm run preview
```

## Estructura del Proyecto

```
src/
├── components/       # Componentes Vue reutilizables
│   ├── common/      # Componentes genéricos
│   ├── cotizacion/  # Componentes específicos de cotizaciones
│   └── forms/       # Componentes de formularios
├── pages/           # Páginas principales
├── stores/          # Store de Pinia
├── utils/           # Funciones utilitarias
├── types/           # Definiciones TypeScript
├── main.js          # Entrada de la aplicación
├── App.vue          # Componente raíz
└── style.css        # Estilos globales
```

## Herramientas de Desarrollo

- **Linter**: ESLint
- **Formatter**: Prettier
- **Estilos**: Tailwind CSS + PostCSS
- **Build**: Vite
- **Estado**: Pinia

## Comandos Disponibles

- `npm run dev` - Iniciar servidor de desarrollo
- `npm run build` - Build para producción
- `npm run preview` - Preview del build
- `npm run lint` - Validar y corregir código
- `npm run format` - Formatear código con Prettier

## Requisitos

- Node.js >= 18.0.0
- npm >= 8.0.0
